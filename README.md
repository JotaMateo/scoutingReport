# Scouting Report — Coordinador (CFB Dynasty)

Página interactiva tipo sala de visionado para comparar a dos candidatos a coordinador: Teton Saltes (ataque) y Domata Peko (defensa).

- index.html — estructura
- styles.css — estilos
- app.js — interacción
- images/ — fotos recortadas

## Ver en local
Abre index.html en el navegador, o sirve la carpeta con un servidor estático.

## GitHub Pages
Settings -> Pages -> Deploy from branch -> main / root. El archivo .nojekyll evita que Jekyll ignore archivos.
